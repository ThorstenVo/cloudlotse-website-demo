The "connect workflows" diagram — a row of bordered paper nodes joined by thin orange arrows drawn in CSS (never a glyph). Best with 3–5 nodes.

```jsx
<SystemFlow nodes={[
  { tag: "01 / Input", title: "Customer enquiry" },
  { tag: "02 / Context", title: "Project knowledge" },
  { tag: "03 / Processing", title: "Business system" },
  { tag: "04 / Next", title: "Next task" },
]} />
```
