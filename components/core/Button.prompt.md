The CloudLotse call-to-action button — square, uppercase, weight 800, with an optional `→` glyph.

```jsx
<Button variant="primary" arrow href="mailto:hello@cloudlotse.de">Review a workflow</Button>
<Button variant="outline">Contact</Button>
```

Variants: `primary` (orange fill on any surface), `outline` (transparent with a white 72% border — use on dark chrome), `ghost` (ink border on light). Primary hover lightens to `#ff805b` and lifts 2px. Pass `arrow` for the trailing arrow, `disabled` to dim, and `href` to render an anchor.
